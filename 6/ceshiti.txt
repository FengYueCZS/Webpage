6、unsigned int 和 int ，哪个可以表示更大的整数？
答：unsigned int
官方答案：由于默认 int 是 signed 的，即带符号位的整数（可以表示负数），需要用一半的存储空间来存放负数；而 unsigned int 则是完全用于存放正数，因此 unsigned int 可以表示更大的整数。
